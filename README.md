# 중독사회 (Addiction Society)

> 중독을 개인의 도덕 실패가 아닌 **사회 구조적 위험**으로 분석하는 연구·정책·데이터 허브

[![License](https://img.shields.io/badge/license-MIT-green.svg)]()

---

## 📌 프로젝트 소개

**중독사회(Addiction Society)**는 도박·게임·알코올·디지털 중독을 단순한 개인 문제로 보지 않습니다.

불평등, 금융·플랫폼 산업, 도시·노동 구조, AI까지 얽혀 있는 **위험 시스템 전체를 데이터로 읽어내고**, 치유 공동체의 새로운 표준을 제안하는 연구·플랫폼 프로젝트입니다.

### 핵심 명제

```
"중독은 개인의 의지 부족이 아니라, 사회 전체의 구조적 위험 시그널이다."
```

---

## 🎯 프로젝트 단계

| 단계 | 목표 | 상태 |
|------|------|------|
| **Phase 1** | 연구·데이터 허브 | 🟢 진행중 |
| **Phase 2** | 치유 공동체 네트워크 | 🔵 계획 |
| **Phase 3** | AI 기반 상담·정책 실험실 | 🔵 계획 |

## 🔍 주요 기능

### 현재 구현됨 (v1.0)

- ✅ 반응형 랜딩 페이지
- ✅ 12개국 도박정책 비교 프레임워크
- ✅ 연구자료 검색 및 필터링
  - 유형별: 정책 / 연구 / 뉴스 / 기타
  - 도메인별: D0~D3
  - 국가별 필터
  - 태그 다중선택 (AND/OR)
- ✅ articles.json 자동 로드

### 향후 계획

- 🔄 서버 기반 전환 (NestJS + PostgreSQL)
- 🔄 Admin 관리 시스템
- 🔄 AI 자동 요약/분류
- 🔄 AI 상담 시스템

## 🚀 로컬 실행

GitHub Pages로 배포되어 있어 별도 서버 없이 실행 가능합니다.

```bash
# 저장소 클론
git clone https://github.com/[username]/addiction-society.git

# 폴더 이동
cd addiction-society

# 로컬 서버 실행 (Python 3)
python -m http.server 8000

# 브라우저에서 열기
# http://localhost:8000
```
